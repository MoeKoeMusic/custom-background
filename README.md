# MoeKoe 自定义背景插件

## 功能介绍

这是一个为 MoeKoe Music Library 页面开发的简单自定义背景和歌单封面插件。

### 主要功能
- 🖼️ 在个人背景右上角添加上传按钮，点击可更换背景图片
- 📷 在每个歌单封面右上角添加上传按钮，点击可更换封面图片
- 💾 自动保存设置到本地存储
- 🔄 页面刷新后自动恢复自定义图片

## 安装方法

- 1. 放置至MoeKoe Music `plugins` 的插件目录中
- 2. 重启 MoeKoe Music

## 使用方法

1. 安装插件后，访问 MoeKoe Music 的个人中心页面
2. 在个人背景图片的右上角会出现 🖼️ 按钮
3. 在每个歌单封面的右上角会出现 📷 按钮
4. 点击对应按钮选择本地图片文件即可替换
5. 图片会自动保存，下次访问时会自动恢复

## 管理设置

- 点击浏览器工具栏中的插件图标可以打开设置面板
- 在设置面板中可以清除所有自定义设置

## 支持的图片格式

- JPG/JPEG
- PNG  
- GIF
- WebP
- BMP

## 注意事项

- 图片文件会转换为 Base64 格式存储在浏览器本地
- 建议使用文件大小适中的图片以避免性能问题
- 清除数据会导致自定义设置丢失

## 技术实现

- 使用 Chrome Extension Manifest V3
- 通过 FileReader API 读取本地图片文件
- 使用 localStorage 保存图片数据
- 通过 MutationObserver 监听页面变化并自动添加按钮
